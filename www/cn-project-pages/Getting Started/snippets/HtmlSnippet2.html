What is Deployment PackageEditor

Deployment Package Editor let you easily describe and build deployment packages, fully compliant with Deployment Admin Specification (Service Compendium, R4.1, ch. 114).

How to start using Deployment Package Editor

You need to create new Deployment Package Project file. You can either create:

  • Create new file with .dpp extension
  • Use Deployment Package Project wizard: File -> New -> Other... -> OSGi -> Deployment Package Definition
The created file is only a description of the real deployment package. To generate deployment package you can use:
  • Quick Build command from the file context menu
  • Deployment Package Export Wizard (File -> Export... -> Deployment Package Development -> Deployment Package)
