Modeling Deployment Package Content

mToolkit offers a convenient editor for Deployment Package Project files with extension .dpp, namely the Deployment Package Editor. To launch the editor, simply double-click on a selected Deployment Package Project file in one of the navigation views. By default, the Deployment Package Editor opens in the editor area in the Workbench with the Headers tab active, as shown below.
