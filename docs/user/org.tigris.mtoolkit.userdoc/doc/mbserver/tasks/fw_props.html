Configuring the Properties of an OSGi Framework

To configure the properties of an already added OSGi framework:

  1. In the Frameworks view, right-click the framework's node and choose Properties.

    As a result, the Framework Properties dialog appears with the connection settings of the OSGi framework.

  2. Modify their values as when the framework has been added for the first time to the OSGi Frameworks tree (see "Adding/Removing an OSGi Framework").
