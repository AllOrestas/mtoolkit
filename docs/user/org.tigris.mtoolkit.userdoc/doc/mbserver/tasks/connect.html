Connecting to/Disconnecting from an OSGi Framework

Setting Up the OSGi Framework

To be able to connect your OSGi framework to mToolkit and to Eclipse, you have to install the IAgent OSGi bundle in it. Refer to the "Setting up the OSGi Framework" section of this guide.

Connecting and Disconnecting

To connect to an OSGi framework, right-click the framework in the Frameworks view and use Connect Framework.

mToolkit will try to contact the OSGi framework by using its connection settings. On success, mToolkit will change the framework's status icon to connected and will display the bundles, services, deployment packages and applications.

Similarly, to disconnect from the OSGi framework, right-click it in the Frameworks view and select Disconnect Framework or click the Disconnect Framework button in the view toolbar.
