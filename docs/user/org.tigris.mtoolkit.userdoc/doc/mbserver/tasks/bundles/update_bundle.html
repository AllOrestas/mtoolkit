Updating a Bundle

To update an installed bundle on an OSGi framework available in the Frameworks view, execute the next steps:

  1. Select the bundle from the framework's tree, right-click it and invoke Update.
  2. In the displayed dialog, specify the location of the bundle JAR file which contains the update and click OK.

Note: You are not allowed to update system bundles providing vital system functionality - updating some of them might lead to loss of important OSGi framework capabilities.
