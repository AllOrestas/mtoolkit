Starting and Stopping a Bundle

An installed bundle can be started/stopped by using the options of the Frameworks view.

Tip: By selecting several bundles in the framework's tree, you can start/stop all of them with a single operation.

Starting the Bundle

To start a bundle, in the Frameworks view select the bundle in the framework's tree, right-click it and use Start.

Stopping the Bundle

To stop an active bundle, in the Frameworks view select the bundle in the framework's tree, right-click it and use Stop.

Note: You are not allowed to stop certain bundles providing vital system functionality - stopping some of them might lead to loss of important OSGi framework capabilities.
