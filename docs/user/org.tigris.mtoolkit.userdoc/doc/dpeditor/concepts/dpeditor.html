Deployment Package Editor

The Deployment Package Editor of the ProSyst mToolkit is placed at your disposal to ease and automate the process of creating deployment packages in Eclipse. It allows you to modify Deployment Package Project files. These files have extension .dpp and contain information needed to generate a deployment package such as manifest headers, bundle resources and processed resources, certificates and ant build file of the Deployment Package Project file.

Note: Deployment Package Project is not to be confused with project in terms of Eclipse. Under "Deployment Package Project" we mean a special file that is used for designing the content of a deployment package.
